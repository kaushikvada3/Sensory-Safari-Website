# Sensory Safari

**Understand your child's sensory world.**

Sensory Safari is an expert-developed tool designed to identify potential sensory or attentional challenges in children. Just like getting glasses for vision issues, understanding sensory needs is the first step toward better support.

## Live Website

[Visit Sensory Safari](https://anubhuthi.github.io/Sensory-Safari-Website/)

## Features

-   **Liquid Glass UI**: visually stunning and accessible interface.
-   **Adaptive Engine**: adjusts difficulty in real-time.
-   **Sensory Controls**: customizable environments (lights, sounds).
-   **Secure Profile**: data privacy with local profiles.
